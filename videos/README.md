# Vídeos do Empório GranGusto

Esta pasta deve conter os vídeos de apresentação dos vinhos recomendados.

## Formato recomendado
- **Formato:** MP4 (H.264)
- **Resolução:** 720p ou 1080p
- **Duração:** 15-60 segundos
- **Orientação:** Paisagem (horizontal)

## Dica
